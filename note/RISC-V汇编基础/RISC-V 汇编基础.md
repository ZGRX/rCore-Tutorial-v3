推荐从 **RARS** 开始。它是 RISC-V 汇编器、模拟器和运行环境，适合初学者写汇编、单步执行、看寄存器变化。

你可以先掌握这些内容：

```asm
li # 加载立即数  
la # 加载地址  
mv # 寄存器复制  
add # 加法  
ld/sd # 64位读写内存  
lw/sw # 32位读写内存  
beq/bne # 条件跳转  
j # 无条件跳转  
call # 函数调用  
ret # 函数返回
```

然后重点理解这些寄存器：

```asm
zero x0 恒为 0  
ra x1 返回地址  
sp x2 栈指针  
a0-a7 函数参数 / 返回值  
t0-t6 临时寄存器  
s0-s11 被调用者保存寄存器
```
